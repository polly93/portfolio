# Land-Use Recommendation Chatbot

## Overview
This project developed a chatbot that provides soil information and recommends suitable land-use types for farmers. 
Using more than 100,000 soil information records, we trained a deep neural network classification model to predict appropriate land-use categories.


## Project Context
This was a team project completed by 4 members as part of a government-funded big data and deep learning training program in South Korea.


## Motivation
This project was designed to improve access to soil information for older farmers and young people returning to farming, who may face difficulties obtaining relevant agricultural data. By providing soil information efficiently through a chatbot interface, the project aims to support better agricultural decision-making and contribute positively to the development of the agricultural sector.


## Data
The project used soil information records containing soil characteristics and land-use categories. 
The raw data are not included in this repository due to data access limitations.


## Methods
- Data cleaning and preprocessing
- Feature construction
- Deep neural network classification model using Keras
- Model integration into a chatbot interface in collaboration with engineers

## Tools
- Python, R
- Keras
- kakao i open builder
- goormide

## Results
The model generated land-use recommendations based on soil characteristics and was integrated into a chatbot to provide user-facing recommendations.
